# ADM_NUST_MISIS_BPM-24-4_Filatov_Arthur
Репозиторий для сдачи заданий по предмету "Алгоритмы дискретной математики" студента группы БПМ-24-4 Филатова Артура

## Практическая работа 6

Файлы решения:

- `task_2.py` - задание 2, программные представления графа;
- `task_3.py` - задание 3, просмотр дуг, исходящих из вершины;
- `task_4.py` - задание 4, перевод графа из одного представления в другое.

Каждый файл самостоятельный и не использует общие модули.

Вершины графа: `0, 1, 2, 3, 4`.

Дуги графа: `(0, 1), (0, 2), (0, 4), (1, 2), (1, 3), (1, 4), (4, 3), (3, 2)`.

Запуск:

```bash
python task_2.py
python task_3.py
python task_4.py
```
